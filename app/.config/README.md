# Configuration Files

This directory contains all project configuration files for better organization.

## Files

- `postcss.config.js` - PostCSS configuration for CSS processing
- `tailwind.config.js` - Tailwind CSS configuration
- `vite.config.ts` - Vite bundler configuration
- `vitest.shims.d.ts` - Vitest type definitions
- `tsconfig.json` - TypeScript compiler configuration
- `components.json` - ShadCN UI components configuration

## Usage

All configuration files have symbolic links in the project root for compatibility with tools that expect them there.

The actual configuration files are stored here for better organization and separation of concerns.
